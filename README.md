# Calculadora de Tramos Aplicación
⚠️DISPONIBLE SOLO PARA **Android**

🔧**Versión:** 2.0  
Creada con Python y Kivy.   
Esta app es una calculadora en proceso de pruebas, desarrollada por Lucía. ✨   

**Imágenes de la versión 1.0** (desde PC)    


## Características 
- Interfaz sencilla
- Basada en Python + Kivy
- Por ahora **solo disponible en Android**

## Descarga
1. Probar la aplicación directamente:   
👉 **Android v1.02:** https://github.com/LuciaNebula5/Calculadora-de-Tramos/releases/tag/v2.0
👉 **PC:** (próximamente)

2. Hacer la instalación desde código:
   - Clona el repositorio   
   - Instala dependencias: `pip install kivy`  
   - Ejecuta: `python main.py`
