- Interfaz sencilla
- Basada en Python + Kivy
- Por ahora **solo disponible en Android**

## Descarga
1. Probar la aplicación directamente:   
👉 **Android** [Descargar la versión 1.0 desde el Releases]  .......  
👉 **PC** [Descargar la versión 1.0 desde el Releases]  

2. Hacer la instalación desde código:
   - Clona el repositorio   
   - Instala dependencias: `pip install kivy`  
   - Ejecuta: `python main.py`
